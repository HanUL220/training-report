# 최종 프로젝트: MSA 서비스 개발 및 운영환경 구축 (팀 프로젝트)

- **진행 기간:** 2024.10.08 ~ 2024.12.17
- **프로젝트 주제:** AI 기반 숏폼 광고 분석을 통한 의류 이커머스 플랫폼
- **사용 기술:** AI/ML(자연어 처리, 감성 분석), MSA 아키텍처(Spring Boot, API Gateway), React, Docker

## 프로젝트 목표
1. AI 감성분석을 통한 긍정적 상품 리뷰 큐레이션  
2. MSA 아키텍처 기반의 확장 가능한 이커머스 플랫폼 구축  
3. 사용자 맞춤형 쇼핑 경험 제공

## 핵심 기능
1. **AI 기반 숏폼 분석 시스템**
   - 자연어 처리를 통한 리뷰 감성 분석
   - 긍정적인 상품 리뷰 숏폼 자동 선별
2. **인터랙티브 커스텀 피팅룸**
   - 드래그 앤 드롭 방식의 UI
   - 실시간 착용샷 미리보기
   - 개인화된 스타일 추천

## 트러블슈팅
1. **AI 감성분석 정확도 향상**
   - **문제:** 다양한 리뷰 패턴에서 오탐지 많음  
   - **해결:** 데이터 증강, 필터링 로직 추가, 사전 학습된 모델 활용
2. **커스텀 피팅룸 최적화**
   - **문제:** 드래그 앤 드롭 시 렌더링 지연 발생  
   - **해결:** React 컴포넌트 구조 최적화 및 불필요한 리렌더 최소화
3. **애니메이션 성능**
   - **문제:** 프런트 애니메이션이 많은 상황에서 프레임 드롭  
   - **해결:** GPU 가속 CSS 활용, React 라이프사이클 개선

## 프로젝트 성과
- 본 프로젝트는 **최종 프로젝트 부문 대상을 수상**하였습니다.
- **AI 기반 리뷰 분석**으로 신뢰도 높은 상품 정보 제공  
- **MSA 기반** 확장 가능한 서비스 아키텍처 구현  
- **최적화된 애니메이션**을 통한 향상된 사용자 경험 제공
- **인터랙티브 커스텀 피팅룸** 도입으로 사용자 참여도와 체감 만족도 상승

## 느낀 점
- **MSA 아키텍처 이해**: 서비스마다 독립적인 배포·개발이 가능하다는 점이 장점이지만, **서비스 간 통신 방식**(API Gateway, Service Registry 등)과 **데이터 일관성**을 유지하기 위해 더 복잡한 구조와 설계가 필요하다는 것을 알았습니다.
- **AI 기반 리뷰 분석**: 실제 사용자 리뷰 데이터를 처리하며, **불규칙적이고 잡음이 많은 텍스트**를 안정적으로 분석하기 위해서는 도메인 지식, 데이터 전처리, 모델 선택 등 다양한 요소를 고려해야 함을 깨달았습니다.
- **DevOps와 CI/CD**: 여러 마이크로서비스를 관리하고 배포하기 위해 **자동화 파이프라인**(테스트, 빌드, 배포)이 얼마나 필수적인지 체감했고, 장애 상황에서 빠른 롤백과 모니터링의 중요성을 배웠습니다.
- **성능 최적화**: 드래그 앤 드롭, 애니메이션 효과 등 **프런트엔드 사용자 경험**을 강화하려면, 성능을 최적화해야 한다는 점을 알았고, React 라이프사이클이나 GPU 가속 CSS 활용이 매우 큰 영향을 미침을 확인했습니다.
- **인터랙티브 커스텀 피팅룸**: 사용자 참여도를 높이기 위해 **드래그 앤 드롭** 방식을 구현하면서, **UI/UX 설계**가 얼마나 사용자 만족도에 직결되는지 실감했습니다. 실시간 피드백에 따른 **반응형 UI** 설계와 성능 최적화가 중요하다는 점을 배웠습니다.
- **팀워크 강화**: 프로젝트 기간이 길고 범위가 넓어질수록, **명확한 커뮤니케이션**과 이슈 트래킹이 중요하고, 각 모듈 간 **명확한 인터페이스** 정의가 협업의 핵심이라는 사실을 다시금 느꼈습니다.
