# PJT1: Social Media 서비스 개발 (개인 프로젝트)

- **진행 기간:** 2024.07.24 ~ 2024.08.02
- **사용 기술:** React, FormData, Restful API, async/await

## 프로젝트 개요
CRUD 기능을 활용하여 실무형 웹 애플리케이션을 설계 및 구현하는 개인 프로젝트.

## 주요 기능
1. **회원가입 및 로그인**
   - JWT 토큰을 활용한 인증 처리
2. **파일 첨부 및 다운로드**
   - FormData를 이용한 파일 업로드/다운로드 처리
3. **게시물 CRUD**
   - 목록 조회, 상세 조회, 데이터 등록, 수정, 삭제

## 트러블슈팅
1. **React 컴포넌트 기반 개발 적응**
   - **문제:** React가 처음이라 컴포넌트 구조 설계가 낯설었음  
   - **해결:** 공식 문서와 Hook 예제를 참고하며 단계적으로 구축

2. **CRUD 데이터 흐름 개선**
   - **문제:** 데이터 변경 사항이 비동기로 반영되지 않음  
   - **해결:** `setState` 최적화와 `async/await` 활용으로 데이터 흐름 제어

3. **파일 업로드/다운로드 오류**
   - **문제:** 서버와 클라이언트 간 파일 전송 시 CORS 문제 발생  
   - **해결:** FormData 활용, 서버 CORS 설정 보완

## 느낀 점
- **React를 활용한 프론트엔드 개발**: 상태 관리와 컴포넌트 구조 설계를 경험하며, 라이프사이클을 이해하는 것이 얼마나 중요한지 깨달았습니다. 
- **JWT 인증 구현**: 로그인/회원가입 과정을 직접 구현하면서 프론트와 백엔드 간 **보안적인 측면**(토큰 전달, 유효성 검사, 만료 처리 등)에 대해 한층 깊이 이해할 수 있었습니다.
- **파일 업로드/다운로드 프로세스**: FormData와 CORS 문제를 해결하면서, 실제 웹서비스에서 **HTTP 통신**과 **파일 전송**이 어떻게 이루어지는지 체감했습니다.  
- **비동기 로직 설계**: `async/await`를 적극 활용하여 **데이터 흐름 제어**를 배우고, 프로젝트 전반에서 코드 가독성과 유지보수성을 높일 수 있었습니다.
