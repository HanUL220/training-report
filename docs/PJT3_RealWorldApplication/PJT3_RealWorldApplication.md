# PJT3: Real World Application (팀 프로젝트)

- **진행 기간:** 2024.09.23 ~ 2024.10.02
- **사용 기술:** React, Spring Boot, MySQL, Docker, Microservices Architecture

## 프로젝트 개요
AI 기술을 활용한 면접 연습 플랫폼.

## 주요 기능
1. 실시간 화상 면접 연습  
2. AI 기반 답변 분석 및 피드백  
3. 면접 결과 데이터 시각화  
4. 맞춤형 면접 질문 생성  
5. 면접 기록 저장 및 리뷰

## 트러블슈팅
1. **AI 답변 분석 정확도 문제**
   - **문제:** 학습 데이터 부족으로 인한 낮은 정확도  
   - **해결:** 사용자 피드백을 바탕으로 반복 학습, 시각화 대시보드로 개선

2. **면접 데이터 저장 구조**
   - **문제:** 여러 사용자 세션 동시 관리 시 DB 트래픽 증가  
   - **해결:** DB 스키마 및 인덱스 최적화, 필요한 데이터만 선별 저장

## 느낀 점
- **AI와 웹서비스의 접목**: AI 모델을 간단히 활용하는 것만으로는 부족하고, **정확도와 사용자 경험** 모두를 만족시키기 위해서는 지속적인 모델 개선과 UX/UI 설계가 필요함을 깨달았습니다.
- **데이터 처리**: 실시간 면접 세션 기록, 분석 결과 저장 등 **데이터베이스 구조**를 효율적으로 설계하지 않으면 성능 저하나 관리 이슈가 생길 수 있음을 알게 되었습니다.
- **협업과 의사소통**: AI 알고리즘, 프론트엔드, 백엔드 담당이 각각 달랐기 때문에, **서로 다른 영역**을 어떻게 조율하고 통합할지에 대한 소통 능력이 중요하다는 것을 배웠습니다.
